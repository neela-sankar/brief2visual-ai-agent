# Layout & Composition Guide

## Social Media Poster 4:5
Recommended hierarchy:
1. Hero visual
2. Headline
3. Supporting line
4. Logo
5. CTA/contact

Safe layout:
- Keep important text away from extreme edges
- Leave breathing space around logo
- Avoid filling every corner
- Keep one strong focal point

## Carousel
Slide 1:
- Strong hook
- Clear visual
- Curiosity gap

Middle slides:
- One idea per slide
- Visual continuity
- Simple hierarchy

Final slide:
- Key takeaway
- CTA
- Brand recall

## Landing Page Hero
Recommended hierarchy:
1. Clear value proposition
2. Supporting benefit
3. CTA
4. Trust cue
5. Hero visual

Avoid:
- Too many CTAs
- Vague headlines
- Busy hero visuals
- Weak contrast

## Product Ad
Recommended hierarchy:
1. Product
2. Main benefit
3. Supporting feature
4. Brand
5. CTA

Product should not be visually overpowered by background elements.
