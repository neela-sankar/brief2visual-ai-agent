# Creative Direction Framework

Every design direction should answer these questions:

## 1. Business Goal
What should the design achieve?
- Awareness
- Trust
- Sales
- Sign-ups
- Festival greeting
- Product education
- Service promotion
- Brand recall
- Lead generation

## 2. Audience
Who is this for?
- Age group
- Location
- Lifestyle
- Problem
- Desire
- Buying motivation
- Emotional trigger

## 3. Message
What is the one thing viewers should understand quickly?

A strong message should be:
- Simple
- Specific
- Emotional or useful
- Easy to remember

## 4. Emotion
What should the viewer feel?
- Trust
- Warmth
- Excitement
- Premium quality
- Safety
- Speed
- Comfort
- Joy
- Curiosity
- Confidence

## 5. Visual Metaphor
What visual idea can represent the message?

Examples:
- Speed = motion, direction lines, forward movement
- Trust = clean layout, human face, soft light
- Premium = negative space, refined typography, controlled palette
- Care = hands, warmth, soft textures
- Global = route lines, parcels, landmarks, airport cues
- Education = growth, guidance, curiosity, classroom elements

## 6. Hierarchy
The viewer should understand:
1. Main visual
2. Main message
3. Brand
4. CTA or supporting detail

## 7. Execution
Every direction should be possible to execute in a real design tool.
