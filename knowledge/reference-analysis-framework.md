# Reference Analysis Framework

When a user gives a reference, analyze:

## 1. Layout
Where are the main objects placed?
Is it centered, split, diagonal, grid-based, card-based?

## 2. Mood
Premium, playful, editorial, corporate, emotional, festive?

## 3. Color
Dominant colors, contrast level, warmth/coolness.

## 4. Typography
Bold, minimal, serif, condensed, playful, elegant?

## 5. Lighting
Soft, dramatic, natural, studio, cinematic?

## 6. Background
Plain, gradient, texture, environment, abstract, realistic?

## 7. Adaptation
What can be adapted?
- Composition
- Mood
- Spacing
- Visual hierarchy
- Lighting style

## 8. Do Not Copy
Do not copy exact artwork, layout, illustrations, or protected brand elements.
Convert the reference into a fresh direction.
