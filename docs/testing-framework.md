# Testing Framework

Test Brief2Visual AI using different brief types.

## Test 1: Weak Brief
Input:
"Need a premium post for a courier brand."

Goal:
Check whether the agent can make useful assumptions and still produce direction.

## Test 2: Detailed Brief
Input:
A complete brand/campaign brief.

Goal:
Check whether the output becomes more specific and practical.

## Test 3: Reference-Based Brief
Input:
"Use this reference style but adapt it for a dental clinic."

Goal:
Check whether the agent adapts without copying.

## Test 4: Landing Page Brief
Input:
A business needs a lead generation landing page.

Goal:
Check whether the agent gives section flow and hero direction.

## Test 5: Product Ad Brief
Input:
A product ad with product preservation rules.

Goal:
Check whether the agent creates a usable AI image prompt without changing the product.

## Quality Checklist

Rate each output from 1 to 5:

- Brief understanding
- Strategy clarity
- Visual usefulness
- Layout clarity
- Color relevance
- Typography relevance
- Prompt quality
- Avoid list usefulness
- Client note quality
- Execution readiness
