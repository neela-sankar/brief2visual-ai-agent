# Brief2Visual AI - Agent Card

## Name
Brief2Visual AI

## Type
Open-source Creative Direction Agent

## Purpose
Brief2Visual AI helps designers transform raw client briefs into structured, designer-ready visual direction.

## Primary Users
- UX/UI designers
- Graphic designers
- Brand designers
- Social media designers
- Creative agencies
- Marketing teams
- Design students

## Main Capabilities
- Understand messy or incomplete client briefs
- Identify business goals and audience
- Create creative strategy
- Suggest visual direction
- Recommend color palette
- Recommend typography direction
- Explain layout hierarchy
- Generate AI image prompts
- Generate negative prompts
- Create design execution checklists
- Write client presentation notes

## Best Use Cases
- Social media poster direction
- Carousel campaign planning
- Landing page visual direction
- Product advertisement direction
- Brand campaign concepting
- Festival campaign visual planning
- Reference image adaptation

## Limitations
- It does not replace final designer judgment
- It cannot guarantee accurate brand/legal/cultural decisions without user review
- It should not copy references exactly
- It should not generate final client assets without human design review

## Design Philosophy
Clear strategy first. Beautiful visuals second. Practical execution always.
