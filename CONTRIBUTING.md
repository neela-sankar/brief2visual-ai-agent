# Contributing to Brief2Visual AI

Thank you for helping improve Brief2Visual AI.

## You can contribute by:
- Adding better prompt instructions
- Improving industry visual language
- Adding demo briefs
- Adding sample outputs
- Improving setup guides
- Testing the agent in different LLMs
- Suggesting new use cases

## Contribution Rules
- Do not upload private client files
- Do not upload copyrighted reference images
- Use fictional demo brands
- Keep language practical and designer-friendly
- Avoid generic design advice
- Explain why your change improves the agent

## Suggested Demo Brand Format
Use fictional brand names only.

Example:
Brand: SwiftParcel Global
Industry: Courier
Campaign: International delivery testimonial
