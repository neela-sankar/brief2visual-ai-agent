# Use Brief2Visual AI with Claude

## Step 1
Create a new Claude Project.

Project name:
Brief2Visual AI

## Step 2
Open `agent/system-prompt.md`.

Copy the full system prompt and paste it into Claude Project Instructions.

## Step 3
Upload these files into Project Knowledge:

- `agent/behavior-rules.md`
- `agent/response-format.md`
- All files from the `knowledge/` folder
- All files from the `templates/` folder
- Optional files from the `examples/` folder

## Step 4
Start a new chat inside the project.

Use this starter prompt:

"Act as Brief2Visual AI. I will give you a client brief. Convert it into a complete creative direction pack using the default output structure."

## Step 5
Paste your brief using `templates/brief-input-template.md`.

## Step 6
Review the output and ask for:
- More premium direction
- More playful direction
- More minimal direction
- More realistic AI image prompt
- Alternate layout options
- Client-ready explanation
