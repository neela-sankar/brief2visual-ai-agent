# Use Brief2Visual AI with ChatGPT Custom GPT

## Step 1
Open ChatGPT and create a new Custom GPT.

## Step 2
Name it:
Brief2Visual AI

## Step 3
Add this description:
An AI creative direction agent that converts client briefs into visual strategy, layout direction, color palettes, typography guidance, AI image prompts, and client presentation notes.

## Step 4
Copy the content from:
`agent/system-prompt.md`

Paste it into the GPT instructions.

## Step 5
Upload these files as Knowledge:
- `agent/behavior-rules.md`
- `agent/response-format.md`
- All files from `knowledge/`
- All files from `templates/`
- Optional demo files from `examples/`

## Step 6
Enable useful capabilities if available:
- File upload
- Image input
- Web search
- Data/document analysis

## Step 7
Test using the demo briefs.
